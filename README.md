<h1> Introduction</h1>
<ol>
 <li>The global pandemic of COVID-19 has an impact on health care and lifestyle around the world, and early detection is crucial for limiting case spread and mortality. The Reverse transcription Polymerase chain reaction (RT-PCR) is the most widely used diagnosis test; nevertheless, the time and expense of these tests are prohibitive, necessitating the use of additional quick and accessible diagnostic methods.</li>
 <li> An alternative to those test is the Idea of this project</li>
 <li> In this method we utilize Deep Learning Algorithm : CNN ( Convolution Neural Network ) </li>
 <li>We collect a lot of images of X-Rays which are both COVID-19 and Normal Healthy 
X-Rays. We preprocess those Images before diving into the Classification. </li>
 <li>Build a Classifier for detecting COVID and NON-COVID Lung X-Rays</li>
 </ol>
